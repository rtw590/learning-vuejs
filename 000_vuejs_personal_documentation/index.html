Input Binding

add v-model to an element and then you can use that element name in brackets somewhere else. You must define the v-model name in the data section

An example is to put input binding on an input and have that text shown elsewhere like drafting a blog and then you see the live preview of your typing.

Checkbox Binding

If you click the checkbox, it will add the value into the array (blog is a data object and categories is an array inside it)

If you made an array, you can use a v-for to cycle through and display it.
